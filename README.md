# CRUD_C_DadosEmTxt
Projeto final da disciplina de programação I (04/21).

Programa escrito em C, abrangendo boa parte do conteúdo inicial de programação em C (estruturas de repetição e seleção, arrays, strings, structs, ponteiros). 
A persistência de dados foi feita sem integração com banco de dados, eles são salvos em arquivos .txt. Programa está bem primitivo, com muitas melhorias possíveis de implementação.
